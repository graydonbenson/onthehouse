# Live Demo
OnTheHouse was deployed on Firebase and can be accessed [here](https://seng-401-on-the-house.web.app/).

# To Run Locally
1. `cd frontend`
2. `npm i`
3. `npm start`
4. `cd backend/functions`
5. `firebase login`
6. `npm i`
7. `firebase serve -p 5001`
